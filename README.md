# Zampa

Proyecto final de aplicación web para el curso de **(IFCD052PO) Programación en Java** con **Ipartek**.  

La aplicación se desarrolló con **Spring Boot** y las siguientes librerías: 

- **Lombok**: simplica el código con anotaciones
- **Thymeleaf**: motor de plantillas para la presentación de la web
- **JPA/Hibernate**: gestión de la persistencia de datos
- **Spring Boot Devtools**: agiliza el desarrollo de la aplicación

Para el apartado visual (**HTML/CSS**) se ha usado **Bootstrap** para su maquetación y diseño *responsive*.
